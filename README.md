# python-challenge-swapi

A simple application to fetch data from the SWAPI (https://swapi.co), saving them as a CSV file as well.

###### Mandatory requirements
- Django 2.0+
- Python 3.6+
- petl

###### Additional requirements:
- requests
- Pandas
